# 香港政府开放数据平台 - 人口数据集分析报告

> 数据来源：https://data.gov.hk/tc-datasets?category=population
> 数据采集时间：2026年4月21日

## 一、概述

香港政府开放数据平台（DATA.GOV.HK）在"人口"分类下共收录 **225个数据集**，由多个政府机构提供，涵盖人口普查、中期人口统计、人口推算、人口分布、住户统计及专题分析等多类数据。

## 二、数据提供机构

| 机构 | 描述 |
|------|------|
| **政府统计处** (Census and Statistics Department) | 主要数据提供方，负责人口普查及中期人口统计 |
| **规划署** (Planning Department) | 提供人口分布推算及全港人口就业数据矩阵 |
| **社会福利署** | 提供与人口相关的社会服务数据 |
| **民政事务总署** | 提供地区人口统计数据 |

## 四、数据尺度分析

### 4.1 地理空间尺度

香港人口数据的地理空间尺度主要分为以下几个层级：

| 空间尺度 | 数量 | 描述 |
|----------|------|------|
| **次区域** | 5个 | 都會區、新界西北、新界東北、新界東南、新界西南 |
| **区议会分区 (DCD)** | 18个 | 香港十八区 |
| **区议会选区 (DCCA)** | 452个 | 选区级别 |
| **小规划统计区 (STPU)** | 159-211个 | 大合并组/小合并组 |
| **单元区 (SG)** | 1,746-2,342个 | 大合并组/小合并组 |
| **主要屋苑 (MHE)** | 540个 | 统计屋邨 |
| **新市镇** | 多个 | 新界新市镇及新界其他地区 |

### 4.2 统计指标类型

- **人口特征**：总人口、性别分布、年龄结构、种族/民族
- **住户特征**：住户数目、住户规模、家庭类型
- **经济特征**：就业状况、职业分布、收入水平
- **房屋特征**：住房类型、房屋所有权、居住面积
- **教育特征**：教育程度、就学状况
- **迁移特征**：内部迁移、出生地

## 五、数据格式与获取方式

### 5.1 可用数据格式

| 格式 | 说明 | 支持API |
|------|------|---------|
| **CSV** | 表格数据，常用格式 | 部分支持 |
| **XLSX** | Excel格式，保留格式 | 部分支持 |
| **JSON** | API常用格式 | 部分支持 |
| **API** | 空间数据共享平台 | 提供多种格式下载 |

### 5.2 API服务

部分数据集通过**空间数据共享平台 (SDSCP)** 提供API接口，支持多种格式下载：
- GeoJSON
- SHP (ESRI Shapefile)
- KML/KMZ
- GML

### 5.3 典型数据集API格式

```

```

## 八、使用建议

### 8.1 数据选择指南

| 研究目的 | 推荐数据集 |
|----------|-----------|
| 区级人口分析 | 区议会分区统计数据 |
| 细粒度空间分析 | 单元区/小规划统计区数据 |
| 长期趋势分析 | 2001-2021历次普查数据 |
| 未来预测 | 人口分布推算2023-2031 |
| 城市规划参考 | 全港人口及就业数据矩阵 |
| 社会政策研究 | 贫穷情况报告、专题报告 |

### 8.2 数据获取建议

1. **实时数据需求**：优先使用API接口，可获取最新格式的数据
2. **历史数据需求**：下载CSV/XLSX文件，部分数据集提供历史版本下载
3. **空间分析需求**：使用SDSCP平台的GeoJSON或SHP格式数据
4. **多尺度分析**：同一普查年份通常有多个空间尺度的数据集

### 8.3 注意事项

- 部分数据集为一次性发布（如人口普查数据）
- 人口分布推算每1-2年更新一次
- 使用空间数据时需注意地理边界的适用年份
- 部分详细数据可能涉及隐私限制
